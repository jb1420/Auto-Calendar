캘린더 앱에 추가하는 방법
Google Calendar
  1. Google Calendar 웹 접속
  2. 왼쪽 하단 다른 캘린더 + 클릭
  3. 파일에서 가져오기 선택
  4. 다운로드한 .ics 파일 업로드 후 가져오기 클릭
iPhone (Apple Calendar)
  1. .ics 파일을 AirDrop 또는 이메일로 iPhone에 전송
  2. 파일 앱에서 .ics 파일 탭
  3. 캘린더에 추가 선택
Mac (Apple Calendar)
  1. 다운로드한 .ics 파일을 더블클릭
  2. 캘린더 앱이 열리며 가져오기 다이얼로그 표시
  3. 추가 클릭
